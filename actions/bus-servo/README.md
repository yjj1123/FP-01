# 幻尔总线舵机动作组 / Hiwonder Bus-Servo Action Groups

本目录保存机械臂动作组 XML。
This directory stores XML action groups for the robotic arm.

- `init.xml`：初始动作组。
  `init.xml`: initial-position action group.
- `fang.xml`：放置动作序列示例。
  `fang.xml`: example placement action sequence.

动作组类型为 `BUS Servo`，可配合幻尔舵机控制器或主控端串口指令使用。
The action group type is `BUS Servo`, and it can be used with a Hiwonder servo controller or UART commands from the main controller.
